# 📌 Experiment 3.1: ProductCard Component

## 🎯 Aim

To create a reusable ProductCard component using React props.

## 🧠 Objective

* Design a responsive product card
* Use props for dynamic data
* Apply modern CSS styling
* Display stock status conditionally

## ⚙️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* React.js

## 📖 Description

This experiment demonstrates how to build reusable components in React. The ProductCard component receives product details such as name, price, and stock status through props and displays them dynamically.

## 🚀 Features

* Reusable component
* Dynamic data rendering
* Conditional rendering (In Stock / Out of Stock)
* Clean UI design

## 📷 Output

Displays product cards like:

* Wireless Headphones
* Mechanical Keyboard
* Smart Watch

# 📌 Experiment 3.3: Person Class Hierarchy

## 🎯 Aim

To implement a class hierarchy using inheritance for Person, Student, and Teacher.

## 🧠 Objective

* Understand OOP concepts in JavaScript
* Implement inheritance and method overriding
* Display hierarchical data using React

## ⚙️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6 Classes)
* React.js

## 📖 Description

This experiment demonstrates object-oriented programming concepts such as inheritance and polymorphism using JavaScript classes. The Person class acts as a base class, while Student and Teacher extend it.

## 🚀 Features

* Class inheritance
* Method overriding
* Dynamic UI rendering
* Type checking using instanceof

## 🧩 Key Concepts

* Inheritance (`extends`)
* Constructor & `super()`
* Method overriding
* Polymorphism
